Data Structures and Algorithms

Algorithms are a set of instructions that, if executed, solve a particular problem.
Data structures are a way of handling and storing data so that data can be processed
efficiently - often using an algorithm.

Common algorithms and data structures

Search:

A search algorithm is used to retrieve data or information from some data source or data structure.
The term 'search' implies 'retrieving a particular piece of data' - like a phone number.
But it can also refer to solving a particular problem, such as 'what is the most efficient way to travel to x number of points on a map'.

  • linear
  • binary
  • hash

Sort:

A sorting algorithm is used to put elements of a list or other similar data structure in order.
Sorting is often required for other algorithms (such as searching) to work efficiently (or work at all).

  • bubble
  • insertion
  • merge
  • selection

Common data structures

Linear
linked lists, stacks, queues, arrays
Hierarchical
trees, heaps, tries
Other
hashmaps, matrices, graphs
